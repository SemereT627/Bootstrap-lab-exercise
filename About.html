Usage of Flexbox

Flexbox is a CSS display type designed to help craft CSS layouts much easier. It works great responsively.
A Flex property is a flexible property of flexible items. If an item is not flexible, then flex has no effect on it. There are three things need to know while using Flexbox. These are,

  • Flex-grow
  • Flex-shrink
  • Flex-basis

Flex-grow
A flex-grow property is a property that specifies how much the item will grow relative to the rest of the flexible items. It is basically aspects a ratio of value with respect to other flex items. Suppose we want three flex items to be in a menu. The first one with flex-grow 1, the second one flex-grow 2 and the third one is flex-grow 3. what it meant here is that the second item will grow a double size respect to the first one and the third grows triple times of the 1st flex item.
Note: If flex-grow set to 1 it gives equal ratio to the flex items.

Flex-shrink
It is the opposite of the flex-grow property. Here, the larger value of flex-shrink is the quicker to shrink and the smaller value of flex-shrink is the later it shrink.

Flex-basis
This property gives a starting width to the flex item. It is like min-width.

Here is some other property need to know. It is the flex-wrap property.

Flex-wrap
It is a flex property that helps to wrap a flex item when the screen resizes. If the flex-wrap property is not used a content is displayed horizontally and it will have a horizontal scroll bar when screen is small. Since this causes our site to be less responsive, the flex-wrap property helps us in this situation.
